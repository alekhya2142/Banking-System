# Banking-System

📌 Overview
This is a console-based **Banking System** implemented in C++.  
It allows users to:
- View their account balance
- Deposit money
- Withdraw money
- Exit the program safely

The project demonstrates **functions, loops, input validation, and basic financial logic** in C++.

---

## ⚙️ Features
- **Show Balance**: Displays the current account balance with two decimal precision.  
- **Deposit Money**: Adds funds to the account (only positive amounts allowed).  
- **Withdraw Money**: Deducts funds if sufficient balance exists.  
- **Input Validation**: Prevents invalid or negative transactions.  
- **Exit Option**: Ends the program gracefully with a thank-you message.
